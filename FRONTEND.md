# Frontend - Vue 3
- Easy to startup and run on a local machine
- When user logs in, they are presented with a simple dashboard showing a list of recipes, and the ability to search for a recipe
- Allow user to select a recipe to view the ingredients, instructions, and yield info
- Allow the user to edit the recipe
    - Ingredients can be edited in a text box
    - Instructions can be edited in a text box
    - Yield information can be edited in a text box
